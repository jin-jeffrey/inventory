## Technologies Used  
For the backend we used Prisma as our ORM, Express.js for the backend server, and React for the frontend.

## Requirements
* Django

## Quickstart

1. Go into invapp folder
```bash
$ cd inventory\invapp
```
2. Install the python requirements
```bash
$ pip install -r requirements.txt
```
